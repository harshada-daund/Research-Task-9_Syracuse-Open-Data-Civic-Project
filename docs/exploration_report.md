# Exploration Report

## Data Quality
The dataset contains multiple date fields, geographic coordinates, neighborhood labels, and violation descriptions. Key cleaning needs include date parsing, missing-value handling, and derived-field generation.

## Initial Findings
- Violation activity spans multiple years.
- A relatively small set of violation types appears frequently.
- Some neighborhoods have substantially higher violation counts than others.

## Planned Follow-Up
- Compare open vs closed case patterns
- Examine average resolution time
- Improve narrative documentation and validation
