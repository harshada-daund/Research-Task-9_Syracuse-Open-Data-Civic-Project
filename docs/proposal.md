# Project Proposal

## Title
Syracuse Housing Code Violations Dashboard

---

## Summary

This project develops a civic-facing data analysis and dashboard using Syracuse Open Data to examine housing code violations across the city.

The goal is to provide residents, community organizations, and policymakers with clear, data-driven insights into violation trends, neighborhood patterns, case status, and resolution performance.

---

## Problem Statement

Housing code violations impact public safety, housing quality, and neighborhood well-being. However, raw municipal datasets are difficult for non-technical users to interpret.

This project addresses the question:

**How can housing code violation data be transformed into meaningful insights that improve public understanding and support better decision-making?**

---

## Data Sources

- Syracuse Open Data  
- Dataset: Code Violations  

---

## Proposed Solution

The project will build:

- A **data processing pipeline** to clean and analyze the dataset  
- A set of **summary statistics and visualizations**  
- An **interactive Streamlit dashboard** for exploring results  

---

## Key Features

The final dashboard includes:

- Violations by Year  
- Top Violation Types  
- Top Neighborhoods  
- Open vs Closed Cases  
- Average Resolution Time by Neighborhood  
- Data Preview Table  

---

## Deliverables

- Reproducible Python data pipeline  
- Cleaned dataset and summary outputs  
- Interactive Streamlit dashboard  
- Technical documentation  
- Methodology documentation  
- Presentation materials  

---

## Success Criteria

The project is successful if:

- The pipeline runs end-to-end from raw data  
- The dashboard loads and displays correct results  
- Key trends and patterns are clearly visible  
- Documentation is complete and understandable  
- The project provides meaningful civic insights  

---

## Expected Impact

This project helps:
- Residents better understand housing conditions  
- Community groups identify high-impact areas  
- Decision-makers evaluate enforcement and resolution performance  

---

## Summary

This proposal outlines a practical, real-world data project that transforms open municipal data into an accessible and insightful tool for the Syracuse community.
