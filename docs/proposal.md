# Project Proposal

## Title
Syracuse Housing Code Violations Dashboard

## Summary
This project builds a civic-facing dashboard using Syracuse Open Data to help residents and city stakeholders understand neighborhood-level patterns in housing code violations.

## Data Sources
- Syracuse Open Data: Code Violations

## Deliverable
A reproducible analysis pipeline and Streamlit dashboard.

## Success Criteria
- Pipeline runs on raw CSV
- Dashboard loads processed data
- Key charts and summary tables are generated
