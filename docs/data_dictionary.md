# Data Dictionary (Primary Fields)

| Field | Description |
|---|---|
| complaint_address | Address associated with complaint/violation |
| complaint_type_name | Complaint category |
| violation | Violation description/code |
| open_date | Date complaint/case opened |
| violation_date | Date violation recorded |
| comply_by_date | Compliance deadline |
| status_type_name | Current status such as Open or Closed |
| status_date | Status update date |
| Neighborhood | Syracuse neighborhood name |
| Vacant | Vacancy indicator |
| Latitude | Latitude coordinate |
| Longitude | Longitude coordinate |
